## Requirements
- yarn >= 1.22.18
- node >= 16.16.0
- Local mongodb server

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn dev

# production mode
$ yarn prod
```

## Test

```bash
# unit tests
$ yarn test
```

## Additional Info
- Theres a postman collection on the repo to check the endpoints/payloads
